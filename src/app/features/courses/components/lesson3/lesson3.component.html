了解綁定 Binding

在 Angular 模板中，綁定建立了模板創建的 UI 的一部分（DOM 元素、指令或組件）與模型（模板所屬的組件實例）之間的實時連接。此連接可用於將視圖與模型同步，當視圖中發生事件或用戶動作時，或兩者皆有。Angular 的變更檢測算法負責保持視圖和模型同步。

綁定的示例包括：

  • 文本插值 text interpolations
  • 屬性綁定 property binding
  • 事件綁定 event binding
  • 雙向綁定 two-way binding

綁定始終由兩個部分組成：目標（將接收綁定值的對象）和模板表達式（從模型產生值）。
